# FoodHub Project

Restaurant Information System built with Django.

## Installation

1. Clone repository

2. Create virtual environment

python -m venv venv

3. Install dependencies

pip install -r requirements.txt

4. Run migrations

python manage.py migrate

5. Create admin

python manage.py createsuperuser

6. Run server

python manage.py runserver

## Features

- Restaurants catalog
- Restaurant menu
- Booking system
- Reviews and ratings
- User authentication
- Admin panel
- API endpoints
